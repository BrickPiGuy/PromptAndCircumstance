Icebreaker: You Have Been Prompting Longer Than You Think

Bad Prompt

Write something about cybersecurity.

Good Prompt

Explain phishing to a first-year college student using a real-world example in less than 200 words.

Bad Prompt

Generate a dosage calculation worksheet.

Good Prompt

Create a dosage calculation practice worksheet for second-year nursing students covering weight-based IV medications, with 10 problems ranging from beginner to advanced difficulty, and include an answer key.

What Is a Prompt?

A prompt is an instruction that shapes what AI returns.

Google query Voice assistant command YouTube search Chat assistant request

AI is like a very knowledgeable intern: it only works with what you ask for.

Better instructions -> better outputs.

Why Prompting Matters

Prompt 1

Tell me about databases.

Result: Generic and broad.

Prompt 2

Explain relational databases to first-year software development students using one analogy and one real-world example.

Result: Contextual, useful, teachable.

Prompt 3

Write a patient discharge summary.

Result: Generic and unusable.

Prompt 4

Write a discharge summary for a 68-year-old patient recovering from a mild ischemic stroke. Include follow-up instructions, prescribed medications, dietary restrictions, and warning signs that should prompt an ER visit. Use plain language suitable for the patient and family.

Result: Specific, safe, and actionable.

Prompt quality directly influences response quality.

Anatomy of a Great Prompt: T-A-S-K-S

T

Task

Explain, compare, summarize, generate, analyze.

A

Audience

Executive, student, child, customer, developer.

S

Specific Context

Give situation and background details.

K

Keep It Structured

Bullets, table, JSON, markdown, outline.

S

Success Criteria

Length, tone, examples, limits, quality goals.

Prompt Response Refine Improve Finalize
Shorter More formal More examples Simpler language Different audience Add citations

Five Prompt Patterns Everyone Should Know

Explain

Explain recursion using a pizza analogy.

Summarize

Summarize this article in five bullet points.

Role Prompting

Act as a hiring manager.

Transformation

Rewrite this email professionally.

Brainstorm

Generate ten project ideas for beginner programmers.

Common Prompting Mistakes: Spot the Problem

"Help me."

"Teach me programming."

"Write an email to my class."

"Explain cloud, code an app, and make slides in one response."

AI Is Powerful, But Not Perfect

  • Hallucinations
  • Bias
  • Potentially outdated information
  • Overconfident phrasing
  • Privacy concerns

Golden Rule

Verify before you trust.
